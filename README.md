# ブログ記事用リポジトリ
## `site.json`について
```
{
  "title": "タイトル",
  "subtitle": "サブタイトル",
  "permalink": "パーマリンク",
  "content": "Markdownのパス",
  "tags": [
    "タグ配列"
  ],
  "hidden": true 記事を隠すかどうか（Devビルドでは無視されます）
},
```

### 例
```
{
  "title": "宮本フレデリカって最高",
  "subtitle": "レイジーレイジーもすこ",
  "permalink": "imas/frederica-suko",
  "content": "imas/frederica.md",
  "tags": [
    "フレデリカ",
    "アイドルマスター"
  ]
},
```
